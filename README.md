## API Reference

#### Get all items

```http
  GET /https://api.openweathermap.org/data/2.5/weather?q=${cityVal}&units=metric&appid={id}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `id` | `string` | **Required**. Your API key |
